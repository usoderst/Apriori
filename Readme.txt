
The apriori algorithm is a candidate set pruning algorithm. Each time frequent item sets are generated at a given length k, a new candidate set is generated to create the next frequent item sets of length k+1. This candidate set is pruned by removed elements that have a frequency lower then a given minimum support count (typically 1/10 of the number of rows in the database).  There are two versions include, A.py and A_Imroved.py. 

The improved version of Apriori runs faster by removing rows from the database as each candidate set is generated. The removed rows are ones that include no frequent items, and therefore will be involved in generating no new item sets at that candidate set length or any greater candidate set length.
